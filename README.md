# 🕵️ Network Scanner

GUI-сканер портов на Python + PyQt6 + asyncio.

## 🚀 Установка
```bash
pip install -r requirements.txt
python -m app.main

🎯 Возможности

    Сканирование диапазонов портов
    Определение сервисов по баннерам
    Экспорт результатов в CSV
    Асинхронное сканирование (до 1000 одновременных соединений)

⚠️ Предупреждение
Используйте только для аудита собственных сетей.
🛠️ Технологии

    Python 3.10+
    PyQt6, asyncio, pydantic, pandas
