# 🕵️ Network Scanner

GUI-инструмент для аудита сети: сканирование хостов, портов, определение сервисов.

🔧 Стек: Python 3.10+, PyQt6, asyncio, pydantic, logging, pytest

✨ Ключевые фичи:
• Асинхронное сканирование до 1000 одновременных соединений
• Поддержка CIDR, диапазонов, вайлдкардов для гибкого выбора хостов
• Режим "только ping" + опция сканирования портов при недоступном ICMP
• Экспорт результатов в CSV/JSON, цветовая индикация статусов
• Покрытие тестами парсеров ввода, логирование, обработка ошибок

## 🚀 Установка
```bash
pip install -r requirements.txt
python -m app.main

🎯 Возможности

    Сканирование диапазонов портов
    Определение сервисов по баннерам
    Экспорт результатов в CSV
    Асинхронное сканирование (до 1000 одновременных соединений)

🎯 Режимы работы

|    Порты     |    Режим     |                Описание                 |
|--------------|--------------|-----------------------------------------|
| `""` (пусто) | 🔍 Ping only | Проверяет доступность хостов через ICMP |
|  `"80,443"`  | 🚪 Port scan | Сканирует указанные порты               |
|    `"*"`     | 🔥 Full scan | Сканирует все 65535 портов (медленно!)  |

⚠️ Предупреждение
• Проект используется для внутреннего аудита домашней/офисной сети
